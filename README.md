# iyzi-laravel

Iyzico Laravel Package

## Installation


```bash
composer require rodosgrup/iyzi-laravel
```

## Usage

```php
use Rodosgrup\IyziLaravel\IyziLaravel;

$iyzi = new IyziLaravel();
```
